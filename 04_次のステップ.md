## 🛠️ 開発時のヒント

### 1. プレースホルダー画像の使用
開発初期は、以下のサービスでプレースホルダー画像を使用：

```html
<!-- Unsplash Source -->
<img src="https://source.unsplash.com/random/1920x1080?nature" alt="Hero" />

<!-- Placeholder.com -->
<img src="https://via.placeholder.com/1000x1000" alt="Profile" />
```

### 2. Tailwind CSSのカスタムカラー確認
`tailwind.config.js`で定義したカラーを使用：

```jsx
<div className="bg-stone-900 text-stone-50">
  <p className="text-amber-600">アクセントカラー</p>
</div>
```

### 3. Framer Motionのアニメーション確認
`lib/animations.ts`で定義したvariantsを使用：

```jsx
import { fadeInUp } from '@/lib/animations';

<motion.div
  variants={fadeInUp}
  initial="hidden"
  whileInView="visible"
  viewport={{ once: true }}
>
  コンテンツ
</motion.div>
```

### 4. 型定義の活用
`lib/types.ts`の型を使用して型安全な開発：

```typescript
import { Profile, Value, Project } from '@/lib/types';
```

### 5. Gitでのバージョン管理
こまめにコミット：

```bash
git add .
git commit -m "feat: Headerコンポーネント実装"
git push origin main
```

# Sota.N プロフィールサイト - 要件定義書

---

## プロジェクト概要

- **サイト名**: Sota.N Portfolio Site
- **コンセプト**: 「侘び寂び × モダンテクノロジー」に日本庭園テーマを掲げたパーソナルブランディングサイト
- **目的**:
  1. Sota.Nという人物の全体像を伝える
  2. 企業のHPのようなプロフェッショナルな印象を与える
  3. Wabisabi Projectの理念と価値観を体現する
  4. コンテンツ（note・人コマブログ・YouTube）への導線を作る
  5. お問い合わせ窓口を提供する

### 庭園テーマ

サイト全体が「日本庭園を構築していく」イメージで設計される。各ページ・セクションが庭園の一つの要素を担う。

| ページ / セクション | 庭園メタフォア | 実装方法 |
|---|---|---|
| Home Hero | 庭園全体の鳥瞰 | 庭園画像バックグラウンド + スクロール連動パススラックス |
| 座右の銘 | 動かないもの（石・灯籠） | 背景に静的な石の円装飾（blur） |
| 夢・ビジョン | 庭園の色（苴・光） | 底部に苴緑グロウ（blur-3xl） |
| Projects | 積み上がるもの（踏み石） | カード出現時スケールアップアニメーション |
| About Wabisabi | 庭園の雰囲気を繋げる | 同じ苴緑グロウ |

### アニメーション方針

- 全セクション共通: スクロール連動フェードイン（上から動いて現れる）
- Hero: スクロールで画像ズームイン + テキスト上方向に移動しながら消える
- Timeline: デスクトップはホバーポップアップ、モバイルはタップ展開
- Projects: スケールアップ（0.95 → 1）+ フェードイン
- 動きのトーン: 「ゆっくりとした」（ease: [0.22, 1, 0.36, 1]、duration: 0.7s）

### レスポンシブブレークポイント

- モバイル: ~767px（1カラム、ハンバーガーメニュー）
- デスクトップ: md（768px以上）

### プロフィール

- **名前**: Sota.N
- **日本語名**: 楢原 爽太
- **キャッチコピー**: 不完全でも、挑戦し続ける
- **Bio**: 組織を変える。人を支える。DX推進で組織変革をにない、侘び寂びの精神で挑戦していく人を支える世界を作っている。
- **生年**: 1996
- **出身**: 東京
- **障害**: 両耳の軽度聴力障害（障害者手帳6級）

### 価値観（3つ）

1. **身近な人から始める** 🤝
   - 変えたい世界があるなら、まず自分の周りの人にいかにして価値を提供できるか考える。
2. **迷ったら動く** 🚀
   - 完璧な情報がある前にも動く。行動を続けることで、最も理想に近づける。
3. **恩を返す** 🌱
   - 受けた恩を、次の世代へと繋げていく。Give & Givenの循環を続ける。

### 座右の銘（2つ）

1. 「全員を救おうとするな。まず身近にいる人のことを考えてみよう。」
2. 「夢は逃げない。逃げているのは常に自分だ。」

### 夢・ビジョン

- **タイトル**: 夢・ビジョン
- **ビジョン**: 社会福祉×デジタル技術×街づくりを組み合わせた、持続可能な社会モデルを構築する。
- **ゴール**: 土地を買って、夢を持つ人を支える空間を作る。

### Wabisabi Project

- **名前**: Wabisabi Project
- **タグライン**: 不完全な人が挑戦していくことを支えていく
- **Mission**: 不完全を受け入れ、お互いを支え合い、持続可能な未来を築く。
- **Vision**: 「侘び寂び」の精神を根拠にした、挑戦していく人々を支える世界のハブになる。
- **Core Values**:
  1. 共感: 他者の不完全さを理解し、温かい視点で見る。
  2. シンプルさと本質: 複雑さを排し、本質に向き合う。
  3. 不完全への敬意: 完璧を求めず、あるがままの美しさを見出す。
  4. 継続性: 短期的な結果だけでなく、長期的な成長を信じる。
  5. 共同創造: 一人ではなく、みんなで一緒に未来を作る。
- **Principles**: 共創的思考 / 透明な温かさ / 相互支援 / 継続的学びの精神 / 余白の価値
